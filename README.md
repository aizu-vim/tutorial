# チュートリアル関連

[![Build Status](https://travis-ci.org/aizu-vim/tutorial.svg?branch=wip%2Ftravis)](https://travis-ci.org/aizu-vim/tutorial)


## 使用するツールをインストールする

* Ruby をインストールしておくこと

```
$ bundle install
```

## ブログをローカルで確認する

```
$ bundle exec jekyll server
-> 指定されたURLへブラウザからアクセス
```


## スライドをローカルで確認する

```
$ bundle exec slideshow build -o slide _slide
```
